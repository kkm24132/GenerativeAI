
References to a few Open-Source Libraries to Fine-Tune LLMs Locally
- [LLaMA-Factory](https://github.com/hiyouga/LLaMA-Factory)
  - Unified Efficient Fine-Tuning of 100+ LLMs & VLMs (ACL 2024)
  - Best for: UI-based fine-tuning, quick experiments, and multi-model support
  - It is a fine-tuning framework with both CLI and Web UI support.
  - Beginner-friendly, however still powerful enough for serious experiments across many model families.
- [Swift](https://github.com/modelscope/ms-swift)
  - Best for: Large model fine-tuning, multimodal models, Qwen-style workflows, evaluation, and deployment.
  - SWIFT, from the ModelScope community, is a fine-tuning and deployment framework for large models and multimodal models.
- [PEFT](https://github.com/huggingface/peft)
  - Best for: LoRA, adapters, prefix tuning, low-cost training, and efficient model adaptation.
  - PEFT stands for Parameter-Efficient Fine-Tuning.
  - It lets you adapt large pretrained models by training only a small number of parameters instead of the full model.
  - It supports methods such as LoRA, adapters, prompt tuning, and prefix tuning.
